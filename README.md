# MyTaxi – Part 2 React Application

## 1. Public URL of Deployed Application

https://zeyadbatis.github.io/MyTaxi/

## 2. Technology Stack Used

- React 
- TypeScript
- Vite
- React Router DOM
- Lucide React icons
- Browser localStorage
- GitHub Pages for deployment

## 3. How to Run and Build Locally

To run the project locally:

```bash
npm install
npm run dev
```

## 5. Feature Descriptions

### Book Taxi
Customers can create a booking, enter pickup/destination details, choose pickup date/time, and receive a booking reference number.

### Admin Dashboard
Admin can search bookings, view urgent unassigned bookings within 2 hours, and assign available drivers.

### Driver Directory
Admin can view and filter drivers by ID, name, suburb, and availability.

### Track Booking
Customers can enter their booking reference number to view the booking status and assigned driver details.
